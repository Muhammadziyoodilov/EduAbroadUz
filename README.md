## 📖 About

**EduAbroadUz** is a platform for Uzbek students who want to study abroad. Every university listed on the platform goes through a thorough research process by our team — so students only get accurate and verified information. We believe every student deserves honest guidance, not guesswork.

---

## ✨ Features

- 🌐 **3 languages** — Uzbek, Russian, and English
- 🏛️ **Verified universities** — every university is researched by our team before being added
- 📄 **Detailed university pages** — comprehensive info about each university
- 🎲 **Chances Calculator** — a unique calculator for each university to estimate your admission chances
- 💬 **Student reviews** — at least one real comment from a current student of each university
- 🎨 **Modern & clean design** — built for a smooth user experience

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) `v18+`
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/Muhammadziyoodilov/EduAbroadUz.git

# 2. Navigate to the project directory
cd EduAbroadUz

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```
